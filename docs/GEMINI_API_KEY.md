# Gemini API Key Setup

Add this variable to your local `.env` file or EAS environment:

```env
EXPO_PUBLIC_GEMINI_API_KEY=your_gemini_api_key_here
EXPO_PUBLIC_GEMINI_MODEL=gemini-2.5-flash
```

Notes:

- The chatbot screen reads `EXPO_PUBLIC_GEMINI_API_KEY`.
- `EXPO_PUBLIC_GEMINI_MODEL` is optional. The app defaults to `gemini-2.5-flash`.
- Restart Expo after changing `.env` values.
